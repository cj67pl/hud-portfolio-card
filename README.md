**Guild of Architected Systems — HUD Portfolio Card**

**Project Overview**

This project is a futuristic HUD-style portfolio card inspired by video game interfaces. It presents a developer’s profile in a visually immersive way, combining RPG-style stats, interactive skill bars, project showcases, and contact functionality. The design emphasizes game-like UI components while serving as a functional portfolio display.


Note: This HUD-style portfolio card is designed for desktop screens and is not fully responsive on mobile devices. Future updates may include responsive design improvements.


**Features**

•	Developer Info Panel: Displays developer ID, class, guild affiliation, and title.

•	Core Attributes: Animated skill bars for logical reasoning, frontend fundamentals, debugging awareness, and other key development traits.

•	Interactive Pop-ups: Separate panels for guild info, developer bio, skills, projects, and contact form.

•	Project Carousel: Showcase multiple projects with descriptions, technologies used, and outcomes.

•	Tech Track Display: Scrollable track of core tools and technologies (React, CSS, JS, Git, etc.) with tooltip descriptions.

•	Futuristic Design: Neon glow, trapezoid panels, hexagon containers, and parallax-like depth for immersive HUD feel.

•	Modular: Fully structured with HTML, CSS, and JavaScript for easy adaptation and scalability.


**Technologies Used**

•	Frontend: HTML5, CSS3, JavaScript

•	Styling: Custom CSS, Boxicons for icons

•	Optional Enhancements: Can be integrated with React for component-based structure


**Usage**

1.	Clone the repository:
2.	https://github.com/cj67pl/hud-portfolio-card.git
3.	Open index.html in your browser.
4.	Explore interactive panels, skill bars, and project carousel.
